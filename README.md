# 星核 · Orb Core

> **每个人都能拥有一颗私人卫星 —— 它是你的太空 AI 伙伴、你的太空相机、你的永恒太阳。**

`#黑客松巅峰赛` 参赛作品。一场关于"人人都有卫星"的交互式未来体验：用户拥有一颗运行在真实近地轨道上的虚拟私人卫星「星核」，可以用自然语言指挥它拍照、反射阳光、与 AI 对话，甚至用手机实时遥控卫星的姿态。

---

## 产品理念

未来十年，太空将会像今天的云一样触手可及。**星核** 是一个给每个普通人的太空入口：一颗专属于你的智能小卫星，它 24 小时环绕地球，永远被阳光充电，带着一个小小的 AI 住在太空里。

你可以对它说：

> "帮我拍一张深圳湾今天日落的照片。"
> "今晚十点把月光反射到南山公园。"
> "告诉我你现在看到什么。"

它会听懂，执行，并把结果传回你的手机。

### 三大核心能力

| 能力 | 描述 | 用户感受 |
|------|------|----------|
| **太空算力** | 卫星搭载边缘 AI，太阳能 0 成本供电，永不断电 | 你的 AI 伙伴住在卫星上，算力永远在线 |
| **太空相机** | 高分辨率对地拍摄，指定任意坐标成像 | "帮我拍一张家乡此刻的照片" |
| **24 小时太阳** | 反射镜阵列，把阳光定向反射到地面指定位置 | "今晚帮我照亮那场婚礼" |

### 卫星经济系统

太阳能发电 → 算力积分（**Energy Points**）→ 执行任务时消耗：

- 拍一张照片 = 50 EP
- 反射一次阳光 = 100 EP
- 与 AI 对话一轮 = 1 EP

能量不用刻意补充，晒晒太阳就回来了。

### 现实依据

这不是科幻 —— 这三件事今天都已经在发生：

- **太空算力**：欧空局 OPS-SAT 已在国际空间站实测边缘 AI；SpaceX 2026 年起申请部署在轨数据中心。
- **太空相机**：Planet Labs 的 200+ 颗对地观测卫星每日扫描全球，已完全商业化。
- **太空反射阳光**：1993 年俄罗斯 *Znamya 2* 实验成功反射满月亮度的光到地面；Reflect Orbital 2026 年 4 月发射首颗商业反射测试卫星。

星核把这三件事缝合成一个"每个人都能拥有"的产品想象。

## 产品组成

星核是一套 **双端联动** 的体验：**Web 端虚拟任务中心** + **鸿蒙手机地面站**。

```
┌──────────────────────────┐         ┌──────────────────────────┐
│   Web 端                  │ ←──→    │   鸿蒙 App               │
│   虚拟卫星任务中心         │  实时    │   手机地面站             │
│   三维交互 / AI 对话       │  姿态    │   手机即遥控器           │
└──────────────────────────┘         └──────────────────────────┘
```

Web 端负责沉浸式太空体验与展示；鸿蒙端负责让你"举起手机"就能亲手遥控卫星。两端通过实时链路联动 —— 手机转动，卫星同步转动。

## Web 端 · 虚拟任务中心

![ORB CORE Dashboard](./orb.mp4)

一共 5 个独立场景，每个承担不同的叙事角色：

### 主控台 `/`

核心交互页面 —— 一个完整的太空任务中心。

- **全屏三维地球**：真实 NASA 日夜贴图，昼夜过渡、大气辉光、海洋镜面反射、流星、银河带
- **真实轨道卫星**：沿 51.6° 倾角的近地轨道持续运行，带光晕尾迹
- **电影级开场动画**：卫星特写 → 轨道回望 → 主场景，一段 5 秒的过场把你"送上太空"
- **AI 对话窗口**：与星核对话，用自然语言下达任务（拍照 / 反射 / 遥测）
- **实时遥测面板**：经纬高、速度、能量环、子系统状态
- **任务照片查看器**：带 HUD / 扫描线 / 噪点的"卫星拍摄"风格

### 卫星控制台 `SAT CTRL`

主控台左上角按钮展开的 2×2 专业工作台：

```
┌───────────────┬───────────────┐
│  实时遥测      │  姿态控制      │
│  15 项数据     │  3D 可控卫星   │
├───────────────┼───────────────┤
│  直播画面      │  地面轨迹      │
│  4K 模拟推流   │  3 圈脉冲轨道  │
└───────────────┴───────────────┘
```

- **实时遥测**：15 项真实随时间抖动的卫星数据（经纬、速度、温度、能耗、信号 …）
- **姿态控制**：独立 3D 卫星模型，Pitch / Yaw / Roll 滑块实时调整，带推力火焰动画；支持 NADIR / SUN / TARGET / FREE 四种预设模式
- **直播画面**：循环播放卫星视角的深圳 / 上海 / 北京航拍画面
- **地面轨迹**：等距圆柱地图上画出 3 圈轨道的地面投影，带脉冲定位光圈

### 卫星追踪 `/tracker`

致敬 sathunt.com 的专业追踪台，**实时显示 20+ 颗真实卫星**：

- 等经纬世界地图 + **动态昼夜阴影**（根据太阳此刻的位置计算）
- 20 颗真实卫星按类别配色：空间站 / GPS / Starlink / 气象 / 科研 / 其他
- 选中任意卫星 → 显示**完整轨道地面投影** + **90 分钟后预测位置** + **覆盖范围圈**
- **自动追踪模式**：勾选后地图随卫星移动自动居中
- 横向无缝环绕（经度 180° 不再"断"）
- 左侧面板：搜索、分组筛选、20 颗卫星实时经纬高速

从主控台右上角的 **POV VIEW** 按钮旁可以进入。

### 第一视角 `/pov`

站在卫星上向外看的沉浸视角。可以在地球、月球、太阳、深空四个目标之间自由切换，配备电影级 HUD、扫描线、瞄准十字、实时姿态数据。这里有一个入口可以进入卫星追踪页。

### 巅峰赛视觉 `/peak`

致敬比赛主视觉的独立动画 —— 程序化不规则数字山峰、数字气泡点阵、矩阵雨背景、CRT 扫描线、山顶高光。纯粹的视觉作品。

## 鸿蒙手机端 · 地面站

一个跑在 **HarmonyOS 手机** 上的卫星遥控终端，让用户把手机当成真正的地面站使用。

### 功能亮点

- **遥控链路**：通过本地局域网与 Web 端实时同步，一次扫码即可连接
- **姿态遥控**：打开「姿态操控」页面，**手机怎么转，卫星就怎么转** —— 利用手机自身的陀螺仪 / 加速度计实时映射到卫星的 Pitch / Yaw / Roll
- **AR 卫星查看器**：通过手机摄像头，把卫星 3D 模型"放"到桌面上，可以走近、绕到后面、从下面抬头看
- **3D 预览卡片**：在手机上就能看到当前卫星姿态的实时 3D 预览，和 Web 端完全同步
- **锁屏恢复**：手机锁屏 / 息屏后回来，姿态状态自动恢复，不会丢连接

### 典型使用场景

> 你坐在沙发上，用电脑打开星核 Web 端看到你的卫星绕过非洲上空。
> 掏出手机扫一下屏幕上的二维码，手机和卫星就配对了。
> 你举起手机向右转 30°，电脑屏幕上的卫星也缓缓向右侧倾。
> 按一下「对准太阳」，卫星自动转向太阳方位，开始反射阳光。
> 放下手机，用语音说"帮我拍一张现在头顶的照片"，几秒后手机推送了一张。

这是"太空就在你手里"的最直接表达。

## 本地体验

### Web 端

```bash
cd orb-core
npm install
npm run dev
```

打开 http://localhost:3000 即可进入主控台。支持局域网访问（`http://<本机 IP>:3000`），鸿蒙手机可以通过同一局域网连入。

### 鸿蒙端

```
orbcore-app/
```

使用 **DevEco Studio** 打开工程，连接 HarmonyOS 设备（或模拟器），点击运行即可安装到手机。

首次启动进入「遥控连接」页面，输入 Web 端地址（或扫描二维码），连接成功后可以直接使用姿态遥控 / AR 查看器。
